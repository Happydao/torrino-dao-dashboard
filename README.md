Data updates every 24 hours (5:00 CET)

## Data files

- `data.json` is the legacy aggregate file used by the current dashboard and Google Sheet export. Keep its shape stable.
- `data/summary.json` contains the same aggregate values plus exact totals for the richer dashboard.
- `data/tokens.json`, `data/staking.json`, and `data/nfts.json` contain detailed asset-level data generated by the daily workflow.
- `logs/latest.txt` and `logs/latest.json` expose the latest public run logs. Historical logs are pruned to the latest 30 runs.
